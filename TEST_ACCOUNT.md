   - 用同一个问题测试不同模型

## 🔧 如果遇到问题

### 问题1：无法登录
**解决：**
```bash
# 重新创建测试用户
cd backend
npx ts-node src/scripts/seed-test-user.ts
```

### 问题2：余额显示为 0
**原因：** 测试用户未正确创建
**解决：** 运行上述脚本重新创建

### 问题3：AI 对话报错
**原因：** 后端未配置 API Key
**解决：**
1. 编辑 `backend/.env`
2. 至少配置一个 AI 提供商：
   ```env
   OPENAI_API_KEY=your_key_here
   # 或
   DOUBAO_API_KEY=your_key_here
   ```
3. 重启后端服务

### 问题4：统计页面无数据
**解决：** 使用 AI 对话功能几次，会自动生成记录
