1. 项目概览
═══════════════════════════════════════════════════════════════════════════════

项目名称: Workflow Platform (工作流平台)
项目定位: 类似 Coze/Dify 的 AI 工作流可视化编辑平台
当前版本: 1.0.0
项目阶段: 核心功能已完成，测试阶段

技术栈:
  前端: React 19 + TypeScript + Vite + Tailwind CSS + React Flow
  后端: Node.js + Express + TypeScript + Prisma ORM + SQLite/PostgreSQL
  数据库: SQLite (开发) / PostgreSQL (生产)

8. 核心业务流程
═══════════════════════════════════════════════════════════════════════════════

1. 用户注册流程
   注册页面 → POST /api/auth/register → 创建 User → 初始化余额(50K积分) → 返回 JWT

2. AI 对话流程
   输入+模型选择 → POST /api/ai/chat → 调用 AI SDK → 计算成本 → 扣费 → 记录日志 → 返回结果

3. 工作流执行流程
   编辑器 → 运行工作流 → POST /api/workflows/:id/execute → 解析节点 → 按序执行 → 计算成本 → 返回结果

4. 工作包导入流程
   发现工作包 → 点击导入 → POST /api/workspace/import-package → 创建关联 → 更新布局 → 刷新显示
