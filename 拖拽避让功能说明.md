# 工作流画布拖拽避让功能说明

## 功能概述

实现了一个智能的工作流画布拖拽避让系统，支持两级元素（容器和卡片）的智能避让。

## 核心特性

### 1. 两级元素系统

- **容器（Container）**：第一级元素，用绿色虚线框表示（📦图标）
- **卡片（Card）**：第二级元素，即原有的工作流卡片

### 2. 智能避让规则

#### 拖拽容器时
- 所有元素（包括其他容器和卡片）都会自动避让
- 避让范围：全局

#### 拖拽卡片时
- 只有同级的卡片会避让
- 容器不参与避让
- 避让范围：仅限第二级元素

### 3. 避让算法特点

- **碰撞检测**：实时检测矩形碰撞
- **最优方向**：选择重叠最小的方向进行避让
- **连锁反应**：支持最多3轮迭代，处理连锁避让
- **平滑动画**：使用 cubic-bezier 缓动函数，动画时长 0.3s
- **画布约束**：避让后的元素位置始终保持在画布范围内

## 使用方法

### 创建容器

1. 点击右下角的 📦 按钮
2. 新容器会出现在画布左上角
3. 可以拖拽和调整容器大小

### 开关避让功能

1. 点击 🧲/⭕ 按钮切换避让功能
2. 🧲 表示避让功能已启用（紫色背景）
3. ⭕ 表示避让功能已禁用（白色背景）

### 拖拽元素

1. 按住元素并拖动
2. 如果启用了避让功能，其他元素会自动移开
3. 松开鼠标，元素固定在新位置
4. 布局会自动保存到服务器

## 技术实现

### 核心文件

1. **dragAvoidanceUtils.ts**
   - 碰撞检测算法
   - 避让位置计算
   - 智能避让（连锁反应）

2. **WorkspacePage.tsx**
   - 集成避让逻辑到拖拽事件
   - 容器和卡片的渲染
   - 状态管理

### 关键算法

#### 碰撞检测
```typescript
detectCollision(element1, element2)
```
判断两个矩形是否重叠

#### 避让方向计算
```typescript
calculateAvoidanceDirection(collision)
```
计算重叠最小的方向作为避让方向

#### 智能避让
```typescript
calculateSmartAvoidance(draggingElement, allElements, canvasSize)
```
考虑连锁反应的智能避让系统

### 数据结构

```typescript
type CardConfig = {

  position: { x: number; y: number }
  size: { width: number; height: number }

```

## 动画效果

### 拖拽元素
- 无位置过渡动画（跟随鼠标）
- 阴影效果增强

### 被避让元素
- 位置过渡：0.3s cubic-bezier(0.4, 0, 0.2, 1)
- 平滑移动到新位置

## 配置参数

### 避让间隙
```typescript
const gap = 20  // 元素之间的间隙（像素）
```

### 画布尺寸
```typescript
const canvasSize = { width: 2000, height: 2000 }
```

### 迭代次数
```typescript
const maxIterations = 3  // 最多3轮连锁避让
```

## 注意事项

1. **性能优化**：避让计算在拖拽过程中实时进行，大量元素时可能影响性能
2. **布局保存**：拖拽结束后自动保存布局，包含 type 属性
3. **兼容性**：确保旧数据迁移时添加默认 type: 'card'

## 未来优化方向

1. **网格对齐**：添加网格对齐功能
2. **分组移动**：支持选中多个元素一起移动
3. **磁性吸附**：元素靠近时自动吸附对齐
4. **历史记录**：支持撤销/重做操作
5. **性能优化**：使用空间索引（如四叉树）优化碰撞检测
