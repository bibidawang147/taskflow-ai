## 🚀 如何运行填充脚本

### 方法 1: 使用 ts-node 直接运行

```bash
cd backend
npx ts-node prisma/seed-explore-data.ts
```

### 方法 2: 编译后运行

```bash
cd backend
npx tsc prisma/seed-explore-data.ts
node prisma/seed-explore-data.js
```

### 方法 3: 配置 package.json

在 `backend/package.json` 中添加:

```json
{
  "prisma": {
    "seed": "ts-node prisma/seed-explore-data.ts"
  }
}
```

然后运行:

```bash
cd backend
npx prisma db seed
```

## 📝 注意事项

1. **数据库连接**: 确保 `DATABASE_URL` 环境变量已正确配置
2. **依赖安装**: 需要安装 `bcryptjs` 用于密码加密
3. **数据清理**: 如需重新填充，请先清空相关表
4. **用户密码**: 所有测试用户的密码都是 `password123`
5. **积分单位**: 1000 coins = 1 元人民币
